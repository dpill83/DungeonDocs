    <h2 class="monster-section-title" id="section-actions">Actions</h2>
    <div class="monster-section-content">
<p><strong><em>Multiattack.</em></strong> The doppelganger makes two Slam attacks and uses Unsettling Visage if available.</p>
<p><strong><em>Slam.</em></strong> <em>Melee Attack Roll:</em> +6 (with Advantage during the first round of each combat), reach 5 ft. <em>Hit:</em> 11 (2d6 + 4) Bludgeoning damage.</p>
<p><strong><em>Read Thoughts.</em></strong> The doppelganger casts <em>Detect Thoughts</em>, requiring no spell components and using Charisma as the spellcasting ability (spell save DC 12).</p>
<p><strong><em>Unsettling Visage (Recharge 6).</em></strong> <em>Wisdom Saving Throw:</em> DC 12, each creature in a 15-foot Emanation originating from the doppelganger that can see the doppelganger. <em>Failure:</em> The target has the Frightened condition and repeats the save at the end of each of its turns, ending the effect on itself on a success. After 1 minute, it succeeds automatically.</p>
    </div>
